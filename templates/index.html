Supervised Machine Learning to Detect Twitter Bots

This project explores how supervised machine learning can be implemented to detect bots on Twitter. Specifically, the Decision Tree algorithm is used. It achieves an accuracy of approximately 99.8% against the test set. A publicly available dataset from Kaggle was used for both training and testing the model. The dataset can be found here.
