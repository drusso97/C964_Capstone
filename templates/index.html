Supervised Machine Learning to Detect Twitter Bots

This project explores how supervised machine learning can be implemented to detect bots on Twitter. Specifically, the Decision Tree algorithm is used. It achieves an accuracy of approximately 99.8% against the test set. A publicly available dataset from Kaggle was used for both training and testing the model. The dataset can be found here.

How It Works

The model was developed in Python, using Numpy for data manipulation and sklearn for the Machine Learning model. Flask is used to integrate the backend Python code with the frontend web application. A dataset containing 50,000 tweets was used. 40,000 of the tweets were used for training and 10,000 were used in the test set. The model leverage the following features of the data: Retweet Count, Mention Count, Follower Count, and Verification Status. Tweet content and hashtags were originally considered later deemed unnecessary.
